#### what does the main include ?

----
### Installation

via curl

> `bash -c "$(curl -fsSL https://raw.githubusercontent.com/clay-f/dotfiles/master/bootstrap.sh)"`

via wget

> `bash -c "$(wget https://raw.githubusercontent.com/clay-f/dotfiles/master/bootstrap.sh -O -)"`
