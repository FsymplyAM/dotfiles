this is fast configration for RoR, zsh, Vim, etc.

### How does install ?
	* bash dotfiles/bootstraps.sh

### How does it conclude ?
	* zsh
	* vim
	* ruby
	* tmux
### How does it uninstall ?
```shell
    sh -c "$(wget https://raw.githubusercontent.com/lyf-t/dotfiles/master/bootstrap.sh -O -)"
```
